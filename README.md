## A/B гипотеза

**Проблема:** 58,2% пользователей уходят после одобрения карты, не сделав первый перевод.

**Гипотеза:** Добавление интерактивного чек-листа «3 шага до первого перевода» после одобрения карты увеличит конверсию с 41,8% до 55%.

**Ожидаемый эффект:** +132 перевода на каждые 1 000 одобренных карт.
